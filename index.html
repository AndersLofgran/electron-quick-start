Electron Quick Start


Getting started:

  1. Go to: https://github.com/Xadr123/electron-quick-start (Notice that this opens a NEW Electron window using Chromium.)
  2. Fork and Clone the repo into your desired filepath.
  3. cd into the folder you just cloned down.
  4. Run 'npm i -g electron' in your terminal.
  5. Run 'code .' to open the project in VSCode.
  6. Run 'npm i' to install the required packages.
  7. Run 'npm start' to run the app!

Installing a live reloader

  1. Visit: https://ourcodeworld.com/articles/read/524/how-to-use-live-reload-in-your-electron-project Or follow the steps below.
  2. Run npm i electron-reload
  3. Require electron in your main.js file. "const electron = require('electron')"
  4. Immediately under that, put in require('electron-reload')(__dirname, { // Note that the path to electron may vary according to the main file electron: require(`${__dirname}/node_modules/electron`) });
